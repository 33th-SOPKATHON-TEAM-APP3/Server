## 📢 실행 방법
1. git clone을 진행한다.
2. resources 폴더에 application.yaml을 추가한다.
3. build/libs 디렉터리로 이동해 java -jar ios-three-0.0.1-SNAPSHOT.jar 명령어를 실행한다.
